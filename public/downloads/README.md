# Public downloads

Files in this folder are served statically at `https://moolahiq.com/downloads/<filename>`.

## Vetting checklist PDF

Place your checklist PDF here as:

    public/downloads/vetting-checklist.pdf

Once committed and deployed it will be reachable at:

    https://moolahiq.com/downloads/vetting-checklist.pdf

Link that URL from the Beehiiv welcome email so new subscribers receive the checklist.
